Fraud Detection

To identify and analyse characteristics of fraudulent transactions

Overview

Welcome to PanDuh's Fraud Detection Panda project! This project will utilize pandas to explore a dataset that contains fradulent transactions.

Fraud Detection is a commonly employed activity in the financial industry to identify customers who have obtained money illegally.

Aim

To analyse characteristics of fraudulent transactions.

Dataset

Loaded in the table below is the dataset. Take some time to explore the data.

The Fraud Dataset provided contains 500 rows and 7 columns.

The columns are:

1) index: specifies the row ID in dataset

2) type: specifies the kind of transaction made (e.g. PAYMENT, DEBIT, TRANSFER)

3) amount: specifies the amount of money involved in the transaction

4) oldbalanceOrg: displays the original balance of the account

5) newbalanceOrg: displays the new balance of account after transaction

6) oldbalanceDest: displays the old balance of the destination account, which is the account in which money will be transferred to

7) newbalanceDest: displays the new balance of destination account , which is the account in which money has been transferred to

8) isFraud: indicates the possibility that a transaction is fraud where 1 = yes and 0 = no

  • The function .mean() can be called to the dataframe. It will return the mean values of each variable of type numeric and float.
